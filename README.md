# Blog App APIs

Spring Boot REST API project.

## Features
- User CRUD APIs
- Category CRUD APIs
- Exception Handling
- DTO Mapping using ModelMapper
- MySQL Database
- Spring Data JPA

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Run Locally

git clone <repo-url>

cd blog-app-apis

mvn spring-boot:run
